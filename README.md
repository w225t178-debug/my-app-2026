# STATION (V-01) - Industrial Dashboard Prototype

STATION (V-01) は、インダストリアルでミニマルなデザインを特徴とする、生産性向上と環境管理のためのWebダッシュボード・プロトタイプです。

## 概要

このアプリケーションは、作業に集中するための機能と、ハードウェアの計器を思わせる視覚効果を統合しています。モノクロームを基調とした「Stark」スタイルを採用し、実用性と審美性を両立させています。

## 主な機能

- **Chrono-3D Dial**: Three.js を使用した、ドラッグ操作可能な3Dアナログ時計。
- **Ambient Synth**: 集中力を高めるための環境音合成機能（Rain / Cosmic Drone）。
- **System Metrics Matrix**: CPU負荷やメモリ使用率、温度などを模したリアルタイム・シミュレーター。
- **Core Chronology**: ポモドーロ・テクニックに基づいた集中・回復タイマー。
- **Gamified XP System**: タスク完了やタイマー終了によりXPを獲得し、レベルアップするゲーミフィケーション要素。
- **Task & Schedule**: 直感的なタスク管理（Todo）と、タイムライン形式のスケジュール管理。
- **Meteo Sensor**: 各地の気象情報を表示するダッシュボード・ノード。
- **Quick Memo**: ローカルストレージに自動保存される簡易メモ機能。

## 技術スタック

- **Frontend**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS (Stark Industrial Custom Theme)
- **3D Rendering**: Three.js
- **Icons**: FontAwesome 6
- **Typography**: Google Fonts (IBM Plex Mono, Inter, Silkscreen)

## 使い方

1. `index.html` をブラウザで開きます。
2. 作業を開始する際にタイマーをスタートさせ、環境音（Ambient Synth）を必要に応じて有効にします。
3. タスクを完了させることで、システムレベル（XP）を上昇させることができます。
